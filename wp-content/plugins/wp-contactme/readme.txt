== Description ==
This plugin is only supported on PHP 5.2 or greater.

<!--description-->
ContactMe is an easy to use and easy to modify contact form for WordPress. Written to be easy to use to none coders.

You can design many contact forms with various type of data collection including checkboxes ,radio buttons, textboxes textareas and many more and allows you to choose which fields you want to be mandatory.
To prevent SPAM you have the option of choosing between a nothing, a simple CAPTCHA, the more elaborate ReCAPTCHA. You can also log all submissions to a table and download them as a csv file.
<!--description-->
